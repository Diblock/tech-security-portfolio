# Fitt4Sences – Secure Mobile Application

Fitt4Sences is a mobile application focused on health, habits and wellness tracking, designed with a security-oriented architecture.

The project is being developed as part of a multiplatform application development program and aims to combine modern mobile development with secure backend design.

---

## Project Goals

The objective of Fitt4Sences is to create a personal wellness platform that allows users to track habits, health indicators and daily activities while maintaining strong data protection and secure architecture.

Key goals include:

- Secure user authentication
- Privacy-aware data management
- Modern mobile architecture
- Scalable backend infrastructure

---

## Technologies Used

Mobile development:

- Kotlin
- Jetpack Compose
- MVVM architecture
- Android Studio

Backend architecture:

- Supabase
- PostgreSQL
- Secure authentication system
- API-based data communication

---

## Security Design Considerations

The project incorporates several security principles during development:

- Secure API communication
- Authentication and access control
- Database security practices
- Protection of sensitive user data
- Backend security policies

---

## Planned Features

- Habit tracking system
- Health and wellness metrics
- Activity monitoring
- User profile and progress tracking
- Secure cloud data synchronization
